Image

Images can be responsive to fit the parent's width, and also can be displayed as round shaped.

Examples of Images

You can add class responsive to make your image fit the width of container. It's height will get adjusted based on screen size. If you want to display a small size you can provide a value of max-width. To display rounded images add rounded and square class for adjusting equal width and height.
